# Radio Grotesk (web files)

Place **WOFF2** files from your [Pangram Pangram](https://pangrampangram.com/products/radio-grotesk) license here so the app can load them:

- `RadioGrotesk-Regular.woff2` (required for the UI)
- `RadioGrotesk-Bold.woff2` (optional, for stronger emphasis)

After adding files, restart `npm run dev`.

If these files are missing, the site falls back to **Geist Sans**.
